## Typography Hierarchy

```
# Main Heading (2em)
Largest, used at top of pages

## Section Heading (1.5em)
For article sections, major breaks

### Subsection (1.25em)
For sub-topics within articles

Body Text (1rem / 16px)
Readable, comfortable line height

Small Text (.875em / 14px)
Used for metadata, dates, muted text
```

## Responsive Behavior

### Desktop (1200px+)
- Full width content (37em max)
- Navigation: Home | About | Now | Toggle
- Sidebar (future enhancement)
- Large typography

### Tablet (768px - 1199px)
- Content with padding
- Navigation wraps if needed
- Touch-friendly buttons
- Adjusted typography

### Mobile (< 768px)
- Full width with margins
- Stacked navigation
- Touch-optimized
- Smaller font sizes
- Single column layout
